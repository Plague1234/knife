# KNIFE
สคริปต์นี้ยังอยู่ในการทดลอง อาจจะมีบางส่วนที่ไม่เสถียรและยังไม่พร้อมใช้งาน ทางเรากำลังพัฒนาให้ดีที่สุด

# ฟีเจอร์

 - สแกนหาช่องโหว่
 - สแกนพอร์ตแเปิดหรือปิด
 - สแกนหาsubnets
 - อื่นๆอีกมากมาย

# วิธีอัปเดตสคริปต์เมื่อสคริปต์หมดอายุ
```bash
แค่พิมพ์คำว่า "update" หรือ "UPDATE"
sel > update
หรือ
sel > UPDATE
Update Complete :)
```
